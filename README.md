# 🎓 LearnAI - Responsive Landing Page

A modern and responsive landing page for an AI Learning Platform built using **HTML5** and **CSS3**. The project is designed with a clean UI, responsive layout, and modern design principles.

---

## 🚀 Features

- ✅ Responsive Navigation Bar
- ✅ Hero Section
- ✅ Features Section
- ✅ Statistics Section
- ✅ Popular Courses
- ✅ About Section
- ✅ Testimonials
- ✅ Contact Form
- ✅ Footer
- ✅ Smooth Scrolling Navigation
- ✅ Hover Effects
- ✅ Modern Responsive Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
